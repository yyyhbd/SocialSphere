# SocialSphere

SocialSphere — это прототип веб-приложения социальной сети, разработанный в рамках лабораторной работы по настройке рабочего окружения веб-разработчика. Проект демонстрирует создание базовой структуры веб-приложения, настройку инструментов разработки и использование системы контроля версий.

## Описание проекта

Целью проекта является разработка основы социальной сети, предназначенной для взаимодействия пользователей, публикации контента и обмена информацией. В рамках работы была создана структура проекта, подготовлены базовые файлы и настроено окружение для дальнейшей разработки.

## Используемые технологии

HTML — для создания структуры веб-страниц.  
CSS — для оформления пользовательского интерфейса.  
JavaScript — для реализации интерактивности.  
Node.js — для управления зависимостями проекта.  
npm — как менеджер пакетов.  
Git — для контроля версий.  
Visual Studio Code — в качестве среды разработки.  
Figma — для проектирования интерфейса.

## Структура проекта

css — файлы стилей.  
js — JavaScript файлы.
